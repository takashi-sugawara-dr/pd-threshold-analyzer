# PD 年率調整・閾値分析ツール

貸し倒れ確率（PD）の年率調整、閾値最適化、収益試算をインタラクティブに行うStreamlitアプリです。

---

## 機能概要

### 分析タブ

| セクション | 内容 |
|---|---|
| 1. データプレビュー | アップロードしたCSVの先頭50行を表示 |
| 2. 選択データの可視化 | PD分布・経過年数分布・デフォルト内訳 |
| 3. 年率調整 | 累積PDを年率PDに換算（T<1は自動スキップ） |
| 4. 閾値設定と指標算出 | 3方式で閾値を設定、承認率/Precision/Recall/AUCを表示 |
| 5. 混同行列 | TN/FP/FN/TPをヒートマップで表示 |
| 6. 収益試算 | 1件あたり単価 × 混同行列 → Waterfallチャート |
| 7. 最大利益Threshold | 収益を最大化する最適閾値を自動探索 |
| 8. 承認率 vs 利益 トレードオフ | 承認率と収益の関係を曲線で可視化 |
| 9. ROC曲線 | AUCと現在の閾値位置を表示 |
| 10. PD分布と閾値 | Normal/Defaultを色分けし閾値ラインを重ねて表示 |
| 11. サマリ | 主要KPIを一覧表示 |
| 12. レポートダウンロード | サマリCSV・件別スコアCSVをエクスポート |

### 3シナリオ比較タブ

3つの閾値戦略を並べてKPIと収益を比較します。

| シナリオ | 説明 |
|---|---|
| ① PDしきい値直接指定 | モデルスコアに基づく判断 |
| ② 承認率から逆算 | 営業目標（承認件数）からの逆算 |
| ③ 最大利益（自動） | 収益を最大化する閾値を自動算出 |

---

## 年率調整の計算式

$$PD_{annual} = 1 - (1 - PD_T)^{\frac{1}{T}}$$

- $PD_T$：T年間の累積デフォルト率
- $T$：経過年数
- $PD_{annual}$：1年あたりの年率PD

> **T < 1（融資から1年未満）の場合**、指数が 1/T > 1 となりPDが膨らむため、年率換算をスキップし元のPDをそのまま使用します。

---

## 収益試算の定義

| セル | 意味 | 収益/損失 |
|---|---|---|
| **TN** | 承認 & 正常（想定通りの収益） | 収益 |
| **FN** | 承認 & デフォルト（見逃し貸し倒れ） | 損失 |
| **TP** | 否決 & デフォルト（正しく否決） | 機会利得 |
| **FP** | 否決 & 正常（誤って否決） | 機会損失 |

## セットアップ

### 必要環境

- Python 3.9 以上推奨
- Anaconda環境の場合はシステムPythonを使用してください（後述）

### インストール

```bash
pip install -r requirements.txt
```

### 起動

```bash
streamlit run app.py
```

#### Anaconda環境の場合

Anaconda付属のPythonでは依存ライブラリのバージョン衝突が発生することがあります。システムPythonを指定して起動してください。

```bash
/usr/bin/python3 -m streamlit run app.py
```

または付属スクリプトを使用：

```bash
./run.sh
```

## 入力CSVの形式

| 列 | 必須 | 説明 |
|---|---|---|
| 実績デフォルト列 | ○ | 0（正常）または1（デフォルト）の数値 |
| PD列 | ○ | 0〜1の浮動小数点数 |
| 経過年数列 | ○ | 正の数値（年単位） |
| その他の列 | - | 任意（ダウンロード時にそのまま出力） |

列名はサイドバーで自由に選択できます。以下の列名は自動プリセットされます。

- 実績デフォルト：`actual_default`, `default_flag`, `default`
- PD：`pd_score`, `default_prob`, `pd`
- 経過年数：`elapsed_years`, `years_since_orig`, `years`

### サンプルデータの生成

```bash
python generate_sample.py   # sample_data.csv  (2,000行、整数年)
python generate_sample2.py  # sample_data2.csv (3,000行、小数年・T<1含む)
```

アプリ起動時に `sample_data.csv` が同ディレクトリに存在する場合、自動的にプリセットされます。
